# Coco的个人文艺空间 🎨

一个专注于分享自然摄影和名人名言的个人文艺网站，旨在为访客带来心灵的宁静与智慧的启迪。

## 🌟 网站特色

### 核心功能
- **自然摄影分享** - 精选高质量自然风光照片，每日自动更新
- **名人名言展示** - 经典智慧语录，每10分钟实时更新
- **文艺风格设计** - 现代简约的文艺风格界面
- **响应式布局** - 完美适配各种设备

### 技术亮点
- 🏔️ 动态山水背景，营造宁静氛围
- 📸 照片模态框查看，支持大图浏览
- 💭 名言历史记录，追溯今日智慧
- 🔄 智能自动更新，保持内容新鲜
- 📱 移动端优化，触控交互友好

## 🚀 快速开始

### 本地预览
直接打开 `index.html` 文件即可在浏览器中预览网站。

### Vercel一键部署
项目已配置Vercel部署，支持多种方式：

#### 方法一：GitHub部署（推荐）
1. 将项目推送到GitHub仓库
2. 访问 [vercel.com](https://vercel.com)
3. 导入GitHub仓库，自动部署

#### 方法二：拖拽部署
- 将项目文件夹压缩为ZIP
- 在Vercel网站上拖拽上传

详细部署指南请查看 [DEPLOYMENT.md](./DEPLOYMENT.md)

## 🎯 功能详解

### 自然摄影模块
- **来源**: 精选Unsplash高质量自然风光照片
- **更新频率**: 每日凌晨自动更新
- **展示方式**: 网格布局，支持点击查看大图
- **交互功能**: 鼠标悬停显示详情，点击模态框浏览

### 名人名言模块
- **来源**: 经典名人智慧语录
- **更新频率**: 每10分钟自动轮换
- **展示方式**: 卡片式设计，突出显示
- **历史记录**: 保存今日所有展示过的名言

### 自动更新系统
- **照片更新**: 每天00:00自动更换
- **名言更新**: 每10分钟自动轮换
- **倒计时显示**: 实时显示下次更新时间
- **手动刷新**: 支持用户手动刷新内容

## 🛠️ 自定义指南

### 修改个人信息
编辑 `index.html` 文件中的个人介绍部分。

### 更新摄影照片
在 `script.js` 中修改 `naturePhotos` 数组：
```javascript
const naturePhotos = [
    {
        url: '你的图片URL',
        title: '图片标题',
        source: '来源描述'
    },
    // 更多图片...
];
```

### 更新名人名言
在 `script.js` 中修改 `motivationalQuotes` 数组：
```javascript
const motivationalQuotes = [
    {
        text: "你的名言内容",
        author: "作者名"
    },
    // 更多名言...
];
```

### 调整更新频率
修改 `script.js` 中的时间设置：
```javascript
// 照片更新频率（24小时）
const tomorrow = new Date(now);
tomorrow.setDate(tomorrow.getDate() + 1);

// 名言更新频率（10分钟）
let countdown = 10 * 60;
```

### 自定义颜色主题
修改 `styles.css` 中的CSS变量：
```css
:root {
    --primary-color: #你的主色;
    --secondary-color: #你的辅色;
    --accent-color: #你的强调色;
}
```
